# 木漏れ日 — Komorebi

> 葉の隙間から地面に落ちる陽の光を、**物理ベースの畳み込み**でリアルタイムに描く WebGL2 ビジュアライゼーション。

「木漏れ日」（こもれび）は、樹冠の葉のすきまを抜けた太陽光が地面につくる、揺れる光のまだら模様を指す日本語です。本プロジェクトは、その光景をスプライト（光の点をテクスチャで貼る近似）ではなく、ケプラーの「光図形」理論に基づく**太陽円盤の畳み込み**として GPU 上で文字どおり計算し、ブラウザだけで再現します。

## 特徴

- **物理ベース**: 各画素で太陽の視直径 0.53° の円盤上 48 方向をサンプリングし、樹冠を透過したかを積分する。小さな隙間は楕円の太陽像（Sonnentaler）に、大きな開口は隙間そのままの光のプールに、その中間は自然な融合に — すべて畳み込みの物理から自動的に現れる。
- **スプライトを使わない**: 「光の点」を貼り付ける近似をしないため、スポットの重なりによる融合や、隙間の開閉による明滅が原理的に正しく出る。
- **インタラクティブな風**: 突風・枝の揺れ・葉のはためき・隙間の開閉を多重の時間スケールで重ね、光が漂いながら明滅する。
- **ジェスチャ操作**: タップで波紋状の突風、スワイプで風向き変更、長押しで葉をかき分けて光を差し込む。
- **調整パネル**: 風の強さ・樹冠の高さ・葉の密度・太陽高度・俯角をスライダーで操作。
- **アクセシビリティ配慮**: `prefers-reduced-motion` を尊重（カメラ揺れ無効・初期静止）。
- **自動負荷調整**: 起動直後の実測フレームレートに応じてシミュレーション解像度を落とす。
- **依存ゼロ**: ビルド不要。静的ファイル（HTML / CSS / JS）のみ。

## 操作方法

| 操作 | 効果 |
|---|---|
| タップ / クリック | 波紋状の突風が広がり、スポットがざわめく |
| スワイプ / ドラッグ | なでた方向に風向きが変わり、突風が吹く |
| 長押し | 葉がかき分けられ、指の下に光のプールが開く（ドラッグで移動・離すと閉じる） |
| 「調整」ボタン | パラメータパネルの開閉（Esc でも閉じる） |

### 調整パラメータ

| スライダー | 範囲 | 効果 |
|---|---|---|
| 風の強さ | 0–100 | 揺れ・明滅の激しさ |
| 樹冠の高さ | 5–16 m | スポット径とぼけ量（高いほど大きくソフト） |
| 葉の密度 | 40–90 | 隙間の量（疎なら大きな光のプール） |
| 太陽高度 | 35–80° | 楕円の伸長と光の色温度 |
| 見下ろす角度 | 35–90° | 視点の俯角（90° で真上から） |

## ローカルでの実行

ビルド手順はありません。静的ファイルを HTTP で配信するだけです。

```bash
# リポジトリ直下で任意の静的サーバを起動
python3 -m http.server 8000
# → http://localhost:8000/ をブラウザで開く
```

`file://` で直接開くこともおおむね動作しますが、一部ブラウザの制約を避けるため HTTP 配信を推奨します。
